## DEVELOP
**install**
```sh
* install package
$ npm install

* install cli
$ sudo npm install -g angular-cli@1.0.0-beta.25.5
```


**start:**
```sh
$ npm run start
```

**build:**
```sh
* build 
$ npm run build
* replace index.html link
$ npm run build-static
```
