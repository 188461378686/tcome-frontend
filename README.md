## 开发者
**配置环境**
```sh
* 安装依赖
$ npm install

* 安装全局angular-cli
$ sudo npm install -g angular-cli@1.0.0-beta.25.5
```


**开发:**
```sh
* 本地已有服务端运行
$ npm run start

* 本地暂无服务端 使用远程接口开发
$ npm run start_pro
```

**打包:**
```sh
* 仅在服务器运行 本地无需build
$ npm run build
* 将静态文件上传至qiniu静态服务器 并替换链接(需要配置qiniuKey)
$ npm run build-static
```
